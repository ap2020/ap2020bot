## for Beginners
既存の bot のソースコードを読むと何をすればいいかイメージが湧くかもしれません。
自分が作ろうとしているものとやることが近い bot のソースコードを読むのがおすすめです。

ソースコードを読む際は，まず `main` 関数を読んで処理の大まかな流れを掴み，その後各関数を読み進めるとわかりやすいかと思います。
